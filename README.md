# High School Coding for the Technology Student Association (TSA)

This repository contains resources for running a high school Coding competition for the Technology
Student Association (TSA).

## Structure

 *  **DOMJudge:** Resources for running the automated judging system used at the National TSA Coding
    competition, [DOMjudge](https://www.domjudge.org)
     *  **Docker:** Dockerfiles and docker-compose files for easily running an instance of DOMjudge
        on any computer with Docker installed.
     *  **Problems:** Templates and scripts for creating problem archives that can be easily
        imported into DOMjudge.
